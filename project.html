Projects

AWS Cloud Integration and Data Analytics My current project, and also my largest undertaking, is the development of a brand new ticketing and data visualization software for a company that specializes in billing and collections for hundreds of dental and oral surgery offices across the country. The application will feature multiple microservices, all hosted using a variety of AWS cloud technologies (Fargate, EC2, S3 etc.) This will be company's first foray into the cloud, and the first of many services in the works as they continue to build a SaaS implementation of their business model.

Made Utilizing Google Forms and Sheets For the Super Bowl this year, I hosted a pick em competition that included Oklahoma Sports Animal host Mark Rodgers, this application used both Google Forms and Google Sheets to live update scores as the client input them, using macros from Google Apps Script and SQL. This application was hosted on Google App Engine, and will be returning this year with live bi-directional data support using web sockets.

Football Fantasy League Integration Through the ESPN Fantasy API This application combined a transportation engineering firm's three football fantasy leagues and their data into one dashboard using React and Python. The dashboard displays scores and matchups for the current week, as well as a litany of props and statistics each player.
